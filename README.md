# QA Automation Project - API Testing with Cypress

## Overview
API automation framework using Cypress with:
- Schema validation (AJV)
- Negative testing
- Performance checks

---

## 🚀 Setup Instructions

1. Clone repo:
```
git clone <your-repo-link>
cd qa-api-automation
```

2. Install dependencies:
```
npm install
```

3. Run tests:
```
npx cypress open
```
